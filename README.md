# 嵌入式系統專題簡要內容
## 專題內容：
使用嵌入式系統ESP32實作出一個多功能藍芽體徵計，其功能包括SP02、HR、EMG與ECG的實時測量，而測量結果的展示方法則是利用OLED螢幕顯示與Blurteeth通訊傳輸的方法來實現。
